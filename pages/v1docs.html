Minchat API v1

This is the documentation for the Minchat API. If you are on this documentation, it means you already know what the base endpoint is. Just add the following endpoints to the end of /api/v1 to use them. You must also, for every request, fill the Authorization header in your HTTP requests with your token.

POST /SendMessage/<channel>

Sends a message to the channel denoted by a channel id.

Body

Name Type Description
content string The content of the message

Response

The response is a regular message structure.

Name Type Description
id integer The unique message id
content string The content of the message
channel integer The id of the channel the message was sent to
author integer the id of the author of the message
